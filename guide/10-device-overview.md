# 设备介绍

联调前，建议先熟悉设备外观、指示灯、按键和接口。

## 设备外观

![设备整体外观](/images/device-test/device-overview.png)


## 指示灯

| 指示灯 | 说明 |
| --- | --- |
| 绿灯 | 网络状态 |
| 蓝灯 | 蓝牙状态 |
| 红灯 | 电源状态 |
| 闪烁 | 对应能力未连接或正在连接 |
| 常亮 | 对应能力连接正常 |


## 接口和按键

| 项目 | 说明 |
| --- | --- |
| Type-C | 供电和日志输出 |
| 网口 | 有线网络接入 |
| 配网按键 | 长按后清除已保存网络信息并重启，进入 BLE 配网广播模式 |


## 设备能力

设备支持：

- Wi-Fi 联网。
- 网口联网。
- BLE 配网和绑定。
- 蓝牙音频桥接。
- 声网 RTC 语音传输。

## 进入配网模式

长按 Type-C 接口旁边按键，设备会清除已保存网络信息并重启。重启后进入 BLE 配网广播模式，绿灯闪烁表示进入配网模式。

Device ID 格式：

```text
{product_id}-{serial_num}
```

示例：

```text
ACP-SP2617XXXXX1
```
